# 통합단말 신문고

OO은행 영업점 직원용 통합단말의 VOC 시스템을 시연하기 위한 Streamlit 앱입니다. 
가상의 통합단말 업무 화면에서 불편한 사항을 `통합단말 신문고`를 통하여 개선 요청 할 수 있습니다.

## 설치

```bash
python3 -m pip install -r shinmungo/requirements.txt
```

## 실행

```bash
streamlit run shinmungo/app.py
```

최초 실행 시 `./shinmungo/shinmungo.db`가 생성되고, `./data/terminal_synthetic_xlsx/`의 raw 엑셀 3개를 기준으로 시드 데이터가 들어갑니다.

## 아키텍처

```text
통합단말 화면
  -> session_state 화면 컨텍스트
  -> 신문고 모달 접수
      -> 민감정보 경고
      -> 화면 영역 선택 컨텍스트
      -> 임베딩 유사검색
      -> 도움됨 종결/신규접수
      -> 자동분류 + 담당부서 라우팅
      -> Ticket / History / Follower
  -> 관리자
      -> 유사 티켓
      -> RAG 답변 초안
      -> Answer 저장 + 상태 변경 + Notification
  -> 내 접수내역
      -> 진행상황
      -> 유사 사례 처리 결과 요약
      -> 팔로우 알림
  -> 대시보드
      -> SQL/pandas 집계
      -> 중복 클러스터링
      -> 우선순위 계산
```

## 시연 시나리오

<영업점 질의>
1. 앱 실행 후 `통합단말 화면`에서 `111 계좌 + 카드 동시신규` 또는 `827 퇴직연금 보유상품변경`을 선택합니다.
2. 업무 화면 우측 상단의 `통합단말 신문고` 버튼을 클릭해 모달 접수창을 엽니다.
3. 모달 안에서 `기능 지정하기`를 눌러 팝업을 잠시 닫고, 통합단말 화면에서 수정 요청할 업무 영역을 클릭합니다.
4. 팝업이 다시 열리면 선택 영역 칩과 화면번호, 화면명, 업무영역이 자동 표시된 상태를 확인하고 제목/상세 내용을 작성합니다.
5. `접수하기`를 누르면 시스템이 먼저 유사 요청을 확인합니다.
6. 유사 요청이 있으면 대표 카드 3개를 확인하고, 카드별 `이 내용이 도움이 되었어요`로 신규 접수를 중단하거나 팝업 하단의 `새로운 요청으로 접수해주세요`로 새로 접수합니다.
7. 접수 완료 메시지에서 접수번호, 배정 부서, 자동 분류를 확인하고 `닫기`로 업무 화면에 복귀합니다.

<본부부서 관리자 답변>
1. 사이드바 역할을 `관리자`로 바꾼 뒤 `관리자` 페이지에서 선택 영역 상세, 답변 초안을 확인하고 편집해 저장합니다.
2. 관리자 화면에서 상태 변경 또는 AI 분류 보정을 수행합니다.
3. `대시보드`에서 중복 클러스터, 우선순위, 분류/부서/상태 분포를 확인합니다.

## 참고

- 화면 기준: `111 계좌 + 카드 동시신규`, `827 퇴직연금 보유상품변경`
- 담당부서: `111=금융서비스개발부`, `827=퇴직연금사업부`
